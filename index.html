html (hypertext markup language)

It's responsible for structuring the content of web pages. It defines the elements and their arrangement, including headings, paragraphs, lists, links, and more. HTML provides the foundation for web documents, organizing information in a hierarchical manner and facilitating accessibility and search engine optimization.

CSS (casscading style sheets)

It's responsible for styling the visual presentation of web pages. It defines the appearance of HTML elements, including layout, colors, fonts, and other design aspects. CSS allows web developers to create visually appealing and responsive designs, ensuring consistency across different devices and browsers.

JavaScript

It's responsible for adding interactivity and dynamic behavior to web pages. It enables developers to create interactive elements, handle user input, manipulate the DOM (Document Object Model), and communicate with servers asynchronously. JavaScript is essential for creating engaging web experiences, such as animations, form validation, and real-time updates.
